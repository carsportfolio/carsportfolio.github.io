In The Weeds

Role:

Lead Game Designer and Developer

Date:

September 2022

Context:

University of Colorado Boulder

Description:

  "In The Weeds" is a farming-themed Unity mini-game I developed to showcase my growth in game development, with a focus on tilemapping, character sprites, and dynamic particle effects. The primary goal of this project was to bring together all the technical skills we had been learning about to create a visually engaging 2D experience.

Please note that the 2D assets are not mine. This mini-game is a personal interpretation for skill practice.
